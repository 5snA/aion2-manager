### [구성 요소] 로직 (scripts.js)

#### [MODIFY] [scripts.js](file:///c:/Users/Osu/Desktop/아이온2%20게임%20매니저/scripts.js)

## 검증 계획

### 수동 테스트
1.  **멤버십 날짜**:
    - 멤버십 활성화 시 날짜 입력 모달 등장 확인.
    - 27일 전 날짜 입력 시 날짜가 붉게 표시되는지 확인.
    - 날짜 클릭 시 수정 모달이 다시 뜨는지 확인.
    - (시뮬레이션) 28일 경과 시 체크박스가 자동 해제되는지 확인.
2.  **알림 토글**:
    - 알림 끔 설정 시 정해진 시간에 팝업이 뜨지 않는지 확인.
    - 알림 켬 설정 시 팝업이 정상적으로 뜨는지 확인.
3.  **UI 정리**:
    - 상단 바에서 어비스 초기화 버튼이 사라졌는지 확인.
    - 설정 모달에서 "추가 숙제칸 이름 설정"이 제거되었는지 확인.
    - 설정 목록이 2열 그리드로 표시되어 가시성이 개선되었는지 확인.
